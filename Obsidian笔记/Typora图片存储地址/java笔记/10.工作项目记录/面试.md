# list 遍历时怎么删除一个不报错

使用迭代器删除不会报错

```
Iterator<Integer> iterator = arrayList.iterator();
 while (iterator.hasNext()) {
     Integer integer = iterator.next();
     if (integer.intValue() == 5) {
          iterator.remove();
     }
 }
```
