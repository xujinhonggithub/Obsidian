# 微信小程序

## 1. wxs、js使用正则替换字符串

### 1.1 案例

把 2020-8-3 18:18:23 中的 - 替换为 .

#### 1.1.1 js

```javascript
var str ='2020-8-3 18:18:23';
str.replace(new RegExp('-','g'),'.');
或
str.replace(/-/g,'.');
```

#### 1.1.2  微信小程序 wxs

```wxs
var str ='2020-8-3 18:18:23';
str.replace(getRegExp('-','g'),'.');

输出为
2020.8.3 18:18:23
```
