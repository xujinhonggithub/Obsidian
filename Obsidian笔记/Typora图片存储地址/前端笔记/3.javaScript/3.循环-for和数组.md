## 2.4 数组案例 冒泡排序

```
var array = [5,1,3,9,4,8,7,0,6,2];
    // 循环的次数
    for (var i = 0 ; i < array.length; i++){
        // 每次循环交换位置的次数
        for (var j = 0; j < i; j++) {
            if (array[j] > array[i]) {
                var temp = array[i];
                array[i] = array[j];
                array[j] = temp;
            }
        }
    }
    console.log(array);
```
